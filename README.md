# Ghid dezvoltare web front-end

<img width="1378" height="770" alt="from-html-to-react" src="https://gist.github.com/user-attachments/assets/44fda0ae-b0fe-4d40-a398-42de53ba462c" />

Ghid complet de dezvoltare web front-end: De la `HTML` la `React`
Această colecție de articole aplică principiul [Pareto (80/20)](https://en.wikipedia.org/wiki/Pareto_principle) pentru a condensa informațiile esențiale despre dezvoltarea web modernă într-un format succinct și practic. Acoperă fundamentele **HTML5 semantic și accesibilitatea**, sistemele de design CSS cu **tokens** și **responsive design**, până la implementarea aplicațiilor **React** cu arhitectură modulară.

Ghidul este conceput pentru dezvoltatori care doresc să înțeleagă rapid cele mai importante best practices și să construiască aplicații web scalabile și performante, concentrându-se pe 20% din cunoștințe care generează 80% din rezultate.

## Next steps

### Creare proiect local cu Parcel

După parcurgerea acestor materiale, poți trece la implementarea unei aplicații, local, folosind un app bundler eg. [**Parcel** ](https://parceljs.org/getting-started/webapp/)

**Creare proiect**

```sh
# with node/npm installed nodejs.org
npm create parcel react-client myblog

cd myblog

# to start in development mode
npm run dev

# to build
npm run build
```

**Disable type checking**

```ts
/*tsconfig.js*/

/* Type Checking */
"strict": false

/* Adauga */
"allowJs": true
```

urmată de publicarea în cloud prin servicii precum **Cloudflare Pages** sau **AWS Amplify**.
